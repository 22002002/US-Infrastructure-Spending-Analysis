# US-Infrastructure-Spending-Analysis
End-To-End Data Analytics Project  Analyzing US  Infrastructure Spending using  SQL, Data Modeling, and Power BI

# Project Overview
This project analyzes infrastructure spending across US states using SQL and Power BI.

# Tools Used
-MySQL
-SQL
-Power BI
-Data Modeling

# Project Workflow
Raw Data -> SQL cleaning -> EDA -> Business Insight ->Data Modeling ->Power BI Dashboard

# Dashboard Features 
-Total Budget
-Total Funding
-Total Speding
-Budget Variance
-Top Contractors
-Spending By Category
-Project Status Analysis
